# Weather Report Web App

Web App для отображения прогноза погоды в Telegram боте Polyrythms.

## Локальная разработка

```bash
# Клонируем репозиторий
git clone https://github.com/polyrythms/weather-report
cd weather-report

# Запускаем локальный сервер
python3 -m http.server 8000
# или
npx serve .
